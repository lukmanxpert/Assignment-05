Important FAQ

What is the Document Object Model (DOM) ?

The Document Object Model (DOM) is a programming interface for web documents, representing the page's structure as a tree of objects. It allows scripts to dynamically access, modify, and interact with HTML or XML content.


How do you select an element form the DOM?

You can select an element from the DOM using these common methods:
01. document.getElementById('id') – Selects an element by its ID.
02. document.querySelector('.class or #id or tag') – Selects the first matching element by CSS selector.
03. document.querySelectorAll('.class or tag') – Selects all matching elements as a NodeList.


What is the event delegation in the context of the dom and why is it useful?

Event delegation is a DOM technique where a parent element handles events from its child elements using a single event listener. It is useful because it improves performance by reducing the number of event listeners and allows handling events for dynamically added elements.


how do you manipulate an elements attributes and styles using the dom?

You can manipulate an element's attributes and styles using the DOM with the following methods:
Attributes:
01. element.setAttribute('attribute', 'value') – Sets or updates an attribute. 02. element.getAttribute('attribute') – Retrieves an attribute's value. element.removeAttribute('attribute') – Removes an attribute.
Styles:
01. element.style.property = 'value' – Directly modifies an inline CSS style.
